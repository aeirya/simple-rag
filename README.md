# Simple RAG QA System

A small Retrieval-Augmented Generation (RAG) question answering (QA) system built with:
- LangChain
- Ollama
- BM25 retrieval

The repository also includes small multiple-choice QA datasets used for evaluation.

The questions were generated from factual information extracted from selected Wikipedia pages.
Question generation was performed using ChatGPT 5.5.

This project was created for an assignment in the Master's program in Language and Communication Technologies at EHU/UPV.
